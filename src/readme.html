ATP

Wat is het?

Een applicatie om de rankings van professionele tennisspelers weer te geven. Men kan de volledige lijst van de spelers weergeven in tabelvorm gesorteerd op aantal punten.

De gebruiker kan een speler selecteren, wijzigen, toevoegen,- en verwijderen.

Men kan ook de top 3 weergeven en via daar dan, een speler selecteren, de details ervan bekijken en wijzigen.

De data

De spelers zullen komen van een gesimuleerde API onder de vorm van een array van Player objecten. Daarvoor kunnen we gebruik maken van een 'mock-service', in-memory-web-api (zie cursus), dat ons toelaat http requests te vangen. Op die manier zal de gebruiker spelers kunnen aflijsten, wijzigen, toevoegen en verwijderen.
